SUMMARY
CareerLens AI is a full-stack web application designed to help students evaluate placement readiness, identify skill
gaps, discover suitable career paths, analyze resumes, and receive AI-generated career guidance.

PROJECTS

DATABASE DESIGN

Tables: profiles, skills, projects, certifications, resumes, companies, role_matches, skill_gaps, roadmaps, applications,
ai_chat

AUTHENTICATION FLOW
Methods: Email Signup, Email Login, Google OAuth. Flow: User Signup → Supabase Auth → Create Profile →
Dashboard Redirect. Protected Routes: Dashboard, Profile, Roadmap, Resume, Applications, AI Coach

ROLE MATCHING ENGINE
Inputs: Skills, Projects, Certifications, CGPA. Processing: Weighted Scoring Model. Example: Python = 15 Points,
React = 10 Points, ML Project = 20 Points, Certification = 10 Points. Output: AI Engineer = 82%, Frontend Developer
= 76%, Backend Developer = 71%

COMPANY ELIGIBILITY ENGINE
Inputs: CGPA, Skills, Projects, Certifications. Checks: CGPA Match, Skill Match, Project Match. Formula: Eligibility
Score = (CGPA Weight) + (Skill Weight) + (Project Weight). Output: Eligible, Partially Eligible, Not Eligible

SKILL GAP ANALYZER
Input: Target Role. Compare: User Skills vs Required Skills. Output: Missing Skills. Priority: High, Medium, Low

AI ROADMAP GENERATOR
Input: Current Skills, Target Role. Prompt Structure: Analyze student profile, Generate roadmap, Include weekly
milestones. Output: JSON Roadmap stored in roadmaps table

RESUME ANALYZER
Supported Files: PDF, DOCX. Flow: Upload Resume → Extract Text → Send To Claude → Analyze Resume →
Generate ATS Score. Output: Strengths, Weaknesses, Suggestions

AI CAREER COACH
Chat Interface with context including Skills, Projects, Resume, Target Role. Features: Career Advice, Interview
Guidance, Learning Suggestions. Chat history stored

APPLICATION TRACKER
Statuses: Wishlist, Applied, OA Scheduled, Interview Scheduled, Selected, Rejected. CRUD Operations: Create, Read,
Update, Delete

DASHBOARD ANALYTICS
Widgets: Readiness Score, Role Matches, Applications, Resume Score, Roadmap Progress, Skill Progress. Charts:
Line Chart, Bar Chart, Donut Chart

FILE STORAGE
Supabase Buckets: resumes, avatars, certificates. Rules: Authenticated Access Only
