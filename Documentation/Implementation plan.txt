#### AI Recommendation Service

##### [NEW] [recommendationService.js](file:///c:/Users/Mayan/Projects/CareerLens/server/src/services/recommendationService.js)
Rule-based career recommendation engine:
1. Takes assessment scores (interest, aptitude, personality, technical, communication, leadership, creativity)

3. Generates match percentages for each career
4. Returns top 5-10 career recommendations with match %, salary range, growth rate, difficulty
5. Generates personalized roadmap for selected career
